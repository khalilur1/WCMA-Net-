By default, the scripts do not fix random seeds.
If you need strictly reproducible results, you can add manual seed settings at the top of the scripts:

import random
import numpy as np
import torch

seed = 42
random.seed(seed)
np.random.seed(seed)
torch.manual_seed(seed)
torch.cuda.manual_seed_all(seed)
torch.backends.cudnn.deterministic = True
torch.backends.cudnn.benchmark = False
